The function  f ( x ) = 1 3 x 3 + 1 2 x 2 + k x + 5  has a local maximum and a local minimum. The local maximum is at x = − 3 .

Show that k = − 6 .

[5]
a.

Find the coordinates of the local minimum.

[2]
b.

Write down the interval where the gradient of the graph of  f ( x ) is negative.

[2]
c.

Determine the equation of the normal at x = − 2 in the form y = m x + c .
