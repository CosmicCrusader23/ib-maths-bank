The functions f and g are defined for x∈ℝ by fx=6x2-12x+1 and gx=-x+c, where c∈ℝ.

Find the range of f.

[2]
a.

Given that g∘fx≤0 for all x∈ℝ, determine the set of possible values for c.

b.
