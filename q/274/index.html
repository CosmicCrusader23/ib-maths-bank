Let  f ′ ( x ) = 8 x 2 x 2 + 1 . Given that  f ( 0 ) = 5 , find f ( x ) .

Markscheme / solution

attempt to integrate     (M1)

u = 2 x 2 + 1 ⇒ d u d x = 4 x

∫ 8 x 2 x 2 + 1 d x = ∫ 2 u d u       (A1)

EITHER

= 4 u ( + C )       A1

OR

= 4 2 x 2 + 1 ( + C )       A1

THEN

correct substitution into their integrated function (must have C)       (M1)

5 = 4 + C ⇒ C = 1

f ( x ) = 4 2 x 2 + 1 + 1       A1
