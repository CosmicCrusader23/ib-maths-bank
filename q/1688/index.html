A function is defined by fx=2-12x+5 for -7≤x≤7, x≠-5.

Find the range of f.

[3]
a.

Find an expression for the inverse function f−1(x). The domain is not required.

[3]
b.

Write down the range of  f−1(x).

[1]
c.
Markscheme / solution

f-7= 8 and f7= 1           (A1) 

range is fx≤1, fx≥8           A1A1 


Note: Award at most A1A1A0 if strict inequalities are used.


[3 marks]

a.

interchanging x, y at any stage           (A1) 

y=2-12x+5

12x+5=2-y

122-y=x+5          (A1) 

122-y-5=x

 f−1(x)= 122-x-5  =2+5x2-x          A1


[3 marks]

b.

range is -7≤f-1x≤7,  f-1x≠-5          A1
