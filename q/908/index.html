Consider the matrix A  = ( e x e − x 2 + e x 1 ) , where x ∈ R .

Find the value of x for which A is singular.

Markscheme / solution

finding det A =  e x − e − x ( 2 + e x ) or equivalent         A1

A is singular ⇒ det A = 0         (R1)

e x − e − x ( 2 + e x ) = 0

e 2 x − e x − 2 = 0          A1

solving for  e x          (M1)

e x > 0  (or equivalent explanation)         (R1)

e x = 2

x = ln 2  (only)         A1   N0
