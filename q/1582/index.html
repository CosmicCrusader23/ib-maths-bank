The graph of the function f(x)=ln x is translated by ab so that it then passes through the points (0, 1) and (e3, 1+ln 2) .

Find the value of a and the value of b.

Markscheme / solution

new function is fx-a+b=lnx-a+b            (M1)

f0=ln-a+b=1           A1

fe3=lne3-a+b=1+ln 2           A1

ln-a=lne3-a-ln 2            (M1)

ln-a=lne3-a2

-a=e3-a2

-2a=e3-a

a=-e3  =-20.0855…           A1

b=1-ln e3=1-3=-2            (M1)A1
