u1 + 2d = 1407,  u1 + 9d = 1183   (M1)(A1)

u1 = 1471, d = −32     A1A1

[4 marks]

a.

1471 + (n − 1)(−32) > 0      (M1)

⇒ n < 1471 32 + 1

n < 46.96…      (A1)

so 46 positive terms      A1
