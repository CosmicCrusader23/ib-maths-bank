EITHER

ln x = ln 2 ± ( ln 2 ) 2 + 8 ( ln 2 ) 2 2      M1

= ln 2 ± 3 ln 2 2      A1

( ln x − 2 ln 2 ) ( ln x + 2 ln 2 ) ( = 0 )      M1A1

THEN

ln x = 2 ln 2 or  − ln 2      A1

⇒ x = 4 or  x = 1 2        (M1)A1
