Consider the system of equations A ( x y ) = ( 2 − 3 ) where A  = ( k + 1 − k 2 k − 1 ) and k ∈ R .

Find det A.

[2]
a.

Find the set of values of k for which the system has a unique solution.

[4]
b.
Markscheme / solution

Attempting to find det A              (M1)

det A  = k 2 + 2 k − 1            A1  N2

[2 marks]

a.

System has a unique solution provided det A ≠ 0              (R1)

k 2 + 2 k − 1 ≠ 0               (A1)

Solving  k 2 + 2 k − 1 ≠ 0  or equivalent for k            M1

k ∈ R \  { − 1 ± 2 } ( accept k ≠ − 1 ± 2 , k ≠ − 2.41 , 0.414 )            A1  N3
