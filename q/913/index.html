Given that M = ( 2 − 1 − 3 4 )  and that M2 – 6M + kI = 0 find k.

Markscheme / solution

M2 = ( 2 − 1 − 3 4 ) ( 2 − 1 − 3 4 ) = ( 7 − 6 − 18 19 )           M1A1

⇒ ( 7 − 6 − 18 19 ) − ( 12 − 6 − 18 24 )  + kI = 0         (M1)

⇒ ( − 5 0 0 − 5 )  + kI = 0         (A1)

⇒ k = 5         A1
