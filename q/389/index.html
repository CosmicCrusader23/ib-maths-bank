Let  f ( x ) = tan ( x + π ) cos ( x − π 2 ) where  0 < x < π 2 .

Express  f ( x ) in terms of sin  x and cos  x .

Markscheme / solution

* This question is from an exam for a previous syllabus, and may contain minor differences in marking or structure.

tan ( x + π ) = tan ⁡ x ( = sin x cos x )      (M1)A1

cos ( x − π 2 ) = sin x      (M1)A1

Note: The two M1s can be awarded for observation or for expanding.

tan ( x + π ) = cos ( x − π 2 ) = si n 2 x cos x      A1
