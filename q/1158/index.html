Given that A =  ( 3 − 2 − 3 4 )  and I = ( 1 0 0 1 ) , find the values of λ for which (A – λ I) is a singular matrix.

Markscheme / solution

singular matrix ⇒ det = 0         (R1)

| 3 − λ − 2 − 3 4 − λ |          (A1)

( 3 − λ ) ( 4 − λ ) − 6 = 0          (M1)

⇒ λ 2 − 7 λ + 6 = 0          (A1)

λ = 1 or 6       (A1)(A1)    (C6)

Note: Award (C2) for one correct answer with no working.
