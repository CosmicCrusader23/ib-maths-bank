An archaeological site is to be made accessible for viewing by the public. To do this, archaeologists built two straight paths from point A to point B and from point B to point C as shown in the following diagram. The length of path AB is 185 m, the length of path BC is 250 m, and angle A B ∧ ⁡ C is 125°.

The archaeologists plan to build two more straight paths, AD and DC. For the paths to go around the site, angle B A ∧ ⁡ D is to be made equal to 85° and angle B C ∧ ⁡ D is to be made equal to 70° as shown in the following diagram.

Find the size of angle C A ∧ ⁡ D .
