What is the enthalpy change of reaction for the following equation?

C2H4 (g) + H2 (g) → C2H6 (g)

C2H4 (g) + 3O2 (g) → 2CO2 (g) + 2H2O (l)        ΔH = x

C2H6 (g) + 7 2 O2 (g) → 2CO2 (g) + 3H2O (l)    ΔH = y

H2 (g) + 1 2 O2 (g) → H2O (l)                           ΔH = z

 

A. x + y + z

B. −x − y + z

C. x − y − z

D. x − y + z

Markscheme / solution

D
