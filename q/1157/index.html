If A =  ( x 4 4 2 ) and B =  ( 2 y 8 4 ) , find 2 values of x and y , given that AB = BA.

Markscheme / solution

AB =  ( x 4 4 2 ) ( 2 y 8 4 ) = ( 2 x + 32 x y + 16 24 4 y + 8 )          (A1)

BA =  ( 2 y 8 4 ) ( x 4 4 2 ) = ( 2 x + 4 y 2 y + 8 8 x + 16 40 )          (A1)

AB = BA ⇒ 8 x  + 16 = 24 and 4 y  + 8 = 40

This gives x = 1  and y = 8 .         (A1)  (C3)
