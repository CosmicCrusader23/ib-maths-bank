EXM.1.AHL.TZ0.34

pestleAIHLPaper 1EXM· ahl-1-14-introduction-to-matricessource ↗

The square matrix X is such that X3 = 0. Show that the inverse of the matrix (I – X) is I + X + X2.

Markscheme / solution

For multiplying (I – X)(I + X + X2)       M1

= I2 + IX + IX2 – XI – X2 – X3 = I + X + X2 – X – X2 – X3         (A1)(A1)

= I – X3         A1

= I        A1

AB = I ⇒ A–1 = B        (R1)

(I – X) (I + X + X2) = I ⇒ (I – X)–1 = I + X + X2          AG   N0
