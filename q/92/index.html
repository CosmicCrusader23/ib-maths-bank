Consider the expansion of  ( 2 + x ) n , where  n ⩾ 3 and  n ∈ Z .

The coefficient of  x 3  is four times the coefficient of  x 2 . Find the value of n .

Markscheme / solution

attempt to find coefficients in binomial expansion       (M1)

coefficient of  x 2 : ( n 2 ) × 2 n − 2 ; coefficient of  x 3 : ( n 3 ) × 2 n − 3          A1A1

Note: Condone terms given rather than coefficients. Terms may be seen in an equation such as that below.

( n 3 ) × 2 n − 3 = 4 ( n 2 ) × 2 n − 2        (A1)

attempt to solve equation using GDC or algebraically       (M1)

( n 3 ) = 8 ( n 2 )

n ! 3 ! ( n − 3 ) ! = 8 n ! 2 ! ( n − 2 ) !

1 3 = 8 n − 2

n = 26        A1
