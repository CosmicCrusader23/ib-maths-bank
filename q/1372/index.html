Let A2 = 2A + I where A is a 2 × 2 matrix.

Show that A4 = 12A + 5I.

[3]
a.

Let B =  [ 4 2 1 − 3 ] .

Given that B2 – B – 4I =  [ k 0 0 k ] , find the value of k .

[3]
b.
Markscheme / solution

* This question is from an exam for a previous syllabus, and may contain minor differences in marking or structure.

METHOD 1
A4 = 4A2 + 4AI + I2 or equivalent       M1A1
= 4(2A + I) + 4A + I       A1
= 8A + 4I + 4A + I
= 12A + 5I      AG

[3 marks]

METHOD 2
A3 = A(2A + I) = 2A2 + AI = 2(2A + I) + A(= 5A + 2I)       M1A1
A4 = A(5A + 2I)       A1
= 5A2 + 2A = 5(2A + I) + 2A
= 12A + 5I       AG

[3 marks]

a.

B2 =  [ 18 2 1 11 ]       (A1)

[ 18 2 1 11 ] − [ 4 2 1 − 3 ] − [ 4 0 0 4 ] = [ 10 0 0 10 ]       (A1)

⇒ k = 10      A1
