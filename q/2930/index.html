A ray of monochromatic light is incident on the parallel interfaces between three media. The speeds of light in the media are v1, v2 and v3.

What is correct about the speeds of light in the media?


A.  v3 < v1 < v2

B.  v3 < v2 < v1

C.  v2 < v3 < v1

D.  v2 < v1 < v3

Markscheme / solution

D
